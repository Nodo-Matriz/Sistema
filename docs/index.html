Nodo Matriz sistema de gestión

Sistema interno del Nodo Matriz

Espacio web donde habitan los formularios de gestión de cada edición: actualización de stock por productores, pedidos de consumidores, recepción en galpón, retiro y referidos.

Este sitio no está pensado para navegación general. Cada formulario se accede a través de un link específico que se distribuye en cada momento del ciclo de la edición.

Para productores

Actualización de stock

Cada lunes recibís por WhatsApp un link personalizado a tu form de stock. El link tiene tu identificador (id_productor) ya incluido, así no tenés que escribirlo.

Ver vista previa del form

Para consumidores

Pedido quincenal

Cada quincena se publica el formulario de pedidos con los productos disponibles en esa edición. Lo recibís por WhatsApp y se cierra en la fecha indicada.

El form de pedidos se genera nuevo cada edición, con URL del tipo: forms/pedidos-ED2026-09.html
